// 04 — Tech

How This Was Built
  • IntersectionObserver: Powers the scroll-aware navigation tracking for lower overhead than scroll listeners.
  • JSON Dictionary: Provides instant client-side localization without page reloads.
  • Semantic HTML: Uses native details and summary for accessible, JS-free accordions.
  • CSS Custom Properties: Supports seamless dark mode switching and respects system preferences.
// 05 — FAQ

Frequently Asked Questions

Is this built with a framework?

No. It uses strictly vanilla HTML, CSS, and JavaScript for maximum performance and zero dependency overhead.

How does the language toggle work?

It leverages a simple JavaScript dictionary mapped to data attributes in the HTML. It swaps text instantly without making any network requests.

What about users who prefer less motion?

The site respects the 'prefers-reduced-motion' media query, automatically disabling scroll animations for those who need it.

Are these custom fonts accessible?

Yes. We use Atkinson Hyperlegible, a font specifically designed by the Braille Institute to differentiate common ambiguous characters.
