Frequently Asked Questions

Is this built with a framework?

No. It uses strictly vanilla HTML, CSS, and JavaScript for maximum performance and zero dependency overhead.

How does the language toggle work?

It leverages a simple JavaScript dictionary mapped to data attributes in the HTML. It swaps text instantly without making any network requests.
